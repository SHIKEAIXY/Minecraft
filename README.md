## 游玩注意事项

1.默认游戏模式调成`冒险`

2.难度调成`和平`

3.世界选项只开启`自然生命回复，立即重生`

4.作弊只开启`保留物品栏，启用命令方块`

5.作弊选项中昼夜更替可自行开启或关闭

6.将`生长刻度`改为`0`

7.本地图默认开启防作弊，将无法开启创造模式
 - 可前往`世界重生点后方按钮`中关闭

8.修复闯关游戏失败时，全体玩家受到的影响

9.修复部分存在的Bug & 第一关采用范围存档
